개인정보처리방침

markdown for tistory(이하 “확장 프로그램”)는 사용자의 개인정보를 수집, 저장, 전송 또는 제3자와 공유하지 않습니다.

1. 수집하는 정보
이 확장 프로그램은 티스토리 글쓰기 및 수정 화면에서 사용자가 입력한 제목과 본문 내용을 읽어 실시간 미리보기를 제공할 수 있습니다.
이 정보는 미리보기 기능 제공을 위한 일시적 처리 목적으로만 사용되며, 별도로 저장되거나 외부 서버로 전송되지 않습니다.

2. 정보 이용 목적
확장 프로그램은 다음 목적에 한해 페이지 내 정보를 사용합니다.
- 티스토리 글쓰기/수정 화면에서 실시간 마크다운 미리보기 제공
- 본문과 미리보기 간 스크롤 동기화 제공
- 미리보기 레이아웃 표시

3. 정보 저장 및 전송
확장 프로그램은 사용자의 작성 내용을 별도 데이터베이스, 서버, 외부 API 또는 제3자 서비스로 전송하지 않습니다.
또한 사용자의 개인정보를 수집하거나 장기 저장하지 않습니다.

4. 제3자 제공
확장 프로그램은 어떠한 사용자 정보도 제3자에게 제공하거나 판매하지 않습니다.

5. 권한 사용
확장 프로그램은 티스토리 도메인에서 동작하기 위해 해당 페이지에 접근합니다.
이 접근은 글쓰기/수정 화면에서 미리보기 기능을 제공하기 위한 목적에 한정됩니다.
